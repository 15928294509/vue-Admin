这个项目并没有完全做完，只做了用户管理，角色管理，权限管理和商品管理

技术栈：vue全家桶、thinkjs框架

服务端地址:https://github.com/15928294509/ThinkJS-JWT-

学习目的:
* 重点学习使用JWT在用户和服务器之间传递信息
* 学习Thinkjs的基本运用
* 学习了解vue-element-admin整个构建
后台:mallserver
* 安装依赖npm install
* 运行:npm run start

前端：vue-element-admin
* 安装依赖npm install
* 运行npm run dev
